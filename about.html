Here is an example of JSON:

{"employees":[ {"firstName":"John", "lastName":"Doe"}, {"firstName":"Anna", "lastName":"Smith"}, {"firstName":"Peter", "lastName":"Jones"} ]}

Here is an example of XML:

John Doe Anna Smith Peter Jones Why is it important? Because it can interchange format that is used to transmit serialized information from a website. In other words it makes it easier for humans to read as objects or pairs. -JSON stands for JavaScript Object Notation -JSON is a lightweight data-interchange format -JSON is language independent * -JSON is "self-describing" and easy to understand -JSON can be used with websites, Mobile applications, REST API's, Databases, and more. -JSON is a collection of name value pairs. -JSON can support four primative types: -Strings -Numbers -Booleans -Null * JSON uses JavaScript syntax, but the JSON format is text only, just like XML. Text can be read and used as a data format by any programming language.
